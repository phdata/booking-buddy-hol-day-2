# Code Checker

This skill checks code. Use it to verify that code is correct.

## What to check

- Make sure the code runs
- Check for obvious errors
- Verify the logic makes sense
- Look for potential issues

## How to use

When asked to check code, review it carefully and report any problems you find. Be thorough.

If you find issues, explain what they are and how to fix them.
